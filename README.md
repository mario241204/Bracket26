# Bracket26 - Mundial 2026 PWA

**Bracket26** es una Progressive Web App (PWA) diseñada para seguir de cerca el Mundial de Fútbol 2026. Prioriza una experiencia móvil fluida con un diseño oscuro y moderno.

## 🚀 Características

- **Fase de Grupos**: Clasificación en tiempo real con detalles de partidos por equipo.
- **Fase Eliminatoria**: Visualización interactiva de los cruces desde dieciseisavos hasta la gran final.
- **Sincronización Automática**: Script integrado para obtener resultados y fechas desde una API externa (API-Football).
- **PWA Ready**: Instalable en dispositivos iOS y Android con soporte offline básico.
- **Seguridad**: Configuración protegida y prevención de ataques comunes (SQL Injection, XSS).

## 🛠️ Tecnologías

- **Backend**: PHP 8.x
- **Base de Datos**: MySQL / MariaDB
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
- **API Externa**: API-Football (v3)
